# VOXBULK email logo — copy into all HTML email templates ## Logo URL (production) Use this **HTTPS** URL in every email (never `data:image` base64 — most clients block it): ``` https://api.voxbulk.com/public/brand/logo-black ``` Local dev: ``` http://127.0.0.1:8000/public/brand/logo-black ``` Put files in **`voxbulk-api/logos/`** (not the marketing site or frontend `public/brand/`). Supported names (PNG preferred for email, SVG also works): - `logo-black.png` or `logo-black.svg` - `logo-white.png` or `logo-white.svg` URLs (both work after deploy): ``` https://api.voxbulk.com/public/brand/logo-black https://api.voxbulk.com/public/brand/logo-black.png ``` Check what's on disk: `https://api.voxbulk.com/public/brand` → `"available"` object. After deploy, templates in the database that still contain `data:image` are auto-refreshed on the next API start for `interview_*` keys. --- ## Logo HTML block (paste into any email) ```html ``` --- ## Full email wrapper (recommended) Use `wrap_brand_email()` in Python (`app/data/brand_email_layout.py`) or paste this structure: ```html Your subject
